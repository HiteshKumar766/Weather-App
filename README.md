# Weather App 🌦️

A simple weather application built using **HTML, CSS, and JavaScript** that fetches real-time weather data from the OpenWeather API.

## Features

* Search weather by city name
* Displays:

  * Temperature
  * Humidity
  * Wind Speed
  * City Name
* Dynamic weather icons
* Error handling for invalid city names
* Responsive UI

## Technologies Used

* HTML
* CSS
* JavaScript 
* OpenWeather API




## Setup

1. Create a free account on OpenWeather.
2. Generate an API key.
3. Open `script.js`.
4. Replace:

```js
const apiKey = "PASTE_YOUR_OPENWEATHER_API_KEY_HERE";
```

with

```js
const apiKey = "YOUR_API_KEY";
```

5. Save the file and run `index.html`.


### 4. Run the Project

Open `index.html` in your browser.
